### 예시

**대소문자 바꿔서 출력**

입력:
- 길이 1 이상 20 이하의 영어 알파벳 문자열 s

출력:
- 문자열 s의 각 문자의 대소문자를 반전시킨 문자열

제약 조건:
- 1 ≤ len(s) ≤ 20
- s는 오직 알파벳(A–Z, a–z)으로만 구성
